B站评论区留言抽奖
=================

原本我想给我的粉丝Roll点小玩意，结果发现小UP主没有开启转发动态抽奖的功能，而且B站也没有官方的评论区抽奖的功能，所以就自己做了一个小工具。

## 主要功能
- 获取视频评论区所有用户
- 对评论区用户去重后用random实现随机抽奖


## 运行环境

- [Python 3](https://www.python.org/)

## 第三方库

- [requests](https://pypi.org/project/requests/)
